IntersectionObserver를 이용한 Javascript

1.개요

  • IntersectionObserver를 이용해 사용자가 스크롤시 화면에 요소가 감지되면 실행되는 기능

2-1. IntersectionObserver 생성과 기능 추가

  • IntersectionObserver를 생성하고 observer에 대입합니다.
  • div를 선택자로 가져와 each문을 통해 길이만큼 각 div를 oberser.observe() 안에 매개변수로 지정해 줍니다. 
let observer = new IntersectionObserver(e => {})

let divs = document.querySelectorAll('.test')

divs.forEach((d, i) => {
  observer.observe(divs[i])
})

2-2. IntersectionObserver에 기능 추가

  • test란 클래스의 css에 opacity를 0, transition: all 2s 을 만들어 줍니다.
  • IntersectionObserver에 매개변수를 e로 설정해 forEach를 통해 div안에 있는 요소들의 스타일을 text.isIntersecting -> 요소가 화면에 보일때 로 기준을 나누어 바꿔줍니다, 
/** css 
  .test{
    opacity: 0;
    transition: all 1s;
  }
**/

let observer = new IntersectionObserver(e => {
  e.forEach(text => {
    if (text.isIntersecting) {
      text.target.style.opacity = 1
      text.target.style.transform = 'translateX(0%)'
    } else {
      text.target.style.opacity = 0
      text.target.style.transform = 'translateX(30%)'
    }
  })
})
